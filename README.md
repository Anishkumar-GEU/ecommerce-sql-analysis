## Objective
Design an e-commerce database and perform business analysis using SQL.

- Customers

- Order_Items

## Tools Used

- MySQL
- SQL

## Project Workflow

1. Designed database schema
2. Inserted sample data
3. Performed business analysis using SQL queries
4. Derived actionable business insights

## Business Questions Answered

1. Top selling products
2. Revenue by category
3. Highest spending customers

## Key Insights

- "Python Hoodie" and "AI Nerd T-Shirt" generated highest revenue.
- "DSA Notebook", "SQL Cheat Sheet", "Terminal Stickers" are the most selling products.
- Some categories showed low performance.

## Business Recommendations

- Increase inventory for top-selling products.
- Launch promotions for low-performing categories.
- Focus retention efforts on high-value customers.
